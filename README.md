# mobile-application

## Documentation

Used by [GRPC-spring(v.3.1.0)](https://github.com/grpc-ecosystem/grpc-spring)

## How to run

```shell
git clone https://github.com/Danil-Prog/mobile-application.git
```

Go to root project directory
```shell
chmod +x ./builder.sh
```

```shell
./builder.sh --run
```

Upgrade service on new version
```shell
./builder.sh --upgrade
```
